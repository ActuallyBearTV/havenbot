# Upload This Bot To GitHub

1. Extract this ZIP.
2. Open your GitHub repository: `havenbot`
3. Click **uploading an existing file**.
4. Drag in all files/folders from this extracted project.
5. Click **Commit changes**.

You should see:

```text
src/
package.json
railway.json
Dockerfile
README.md
.env.example
.gitignore
```

Then go back to Railway and click **Refresh**.
